How to Use the Engine

1. Choose Your Input Mode

Select either File Upload or Manual Input from the tabs in the Document Menu.

  • File Upload: Best for existing .docx or .pdf documents. The engine will extract text and headings automatically.
  • Manual Input: Perfect for typing or pasting letters from scratch.

2. Professional Formatting

In Manual Mode, you can use simple shortcuts to style your letter body:

  • Use **text** for bold text.
  • Use *text* for italic text.
  • Lines starting with "To:", "Dear", or "Attention:" are automatically bolded.

3. Signatures & Metadata

  • Reference Number: Enter your official document reference (e.g., SFD/001/2026).
  • Issue Date: Select the date to appear on the letterhead.
  • Signature: Upload a photo or scan of your signature to place it at the end of the document.

4. Processing & Export

Click Process & Preview to generate your multi-page letterhead. Once satisfied:

  • Click Print PDF to download a high-quality PDF with a subject-based filename.
  • Click Print View to use your browser's standard print dialog.
